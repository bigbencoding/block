Code Explained

The forever block loops the code forever, so the program never ends. This makes it so the square can be controlled forever until you kill the program. This is called a loop. Loops run things for a specific amount of repeats or until it meets the requirements. The if block asks "if (statement) than (result)". That means if the statement is true, run the things inside of it. An if block is a type of conditional. A conditional takes a statement and if it makes the requirements, it runs the code inside. We can use this to ask if the users are pressing the arrow keys.
